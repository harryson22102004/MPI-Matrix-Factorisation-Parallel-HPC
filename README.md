# MPI Matrix Factorisation — Parallel HPC
Scaling matrix computations across distributed clusters using Message Passing Interface (MPI).

### 🔬 Research Highlights
- **Domain Decomposition**: Splits large matrices into sub-blocks for parallel processing across multiple CPU nodes.
- **Parallel Communication**: Uses `MPI_Send`, `MPI_Recv`, and `MPI_Bcast` to synchronize data between ranks.
- **HPC Focus**: Optimizes throughput for large-scale scientific simulations.

**Tech Stack:** Python (`mpi4py`), C++.
